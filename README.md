 .NET Core - Tarım Projesi 

Admin Panelli, UI'ın Tamamen Dinamik olduğu ve
Verilerin eklendiği, düzenleyebileceği ve silebileceği, şifresini sıfırlayabileceği bir .NET Core  projesi, yönetici rollunu içerir. 







## Used

- .NET Core 6.0
- Entity Framework
- Identity Framework
- JavaScript, jQuery, C#
- HTML, CSS
